# Asset Files

This directory should contain the following image files for Expo:

- **icon.png** (1024x1024) - App icon
- **splash.png** (1284x2778) - Splash screen
- **adaptive-icon.png** (1024x1024) - Android adaptive icon
- **favicon.png** (48x48) - Web favicon

## Quick Setup

You can generate these using Expo's icon generator or use placeholder images.

### Generate with Expo
```bash
npx expo-app-icon-generator
```

### Use Placeholders
Download simple colored placeholders or create them online.

For now, the app will show warnings about missing assets but will still run fine. You can add proper assets later.
